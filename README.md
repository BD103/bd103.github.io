# [bd103.dev](https://bd103.dev)

This is the source code for my personal website and blog!

## Required Tools

To build this website, you will need:

- [Zola](https://www.getzola.org/) 0.21.0

## Building

You can build this website locally by running:

```shell
$ zola build
```

This site is automatically deployed to Github Pages through [`deploy.yml`](./.github/workflows/deploy.yml).
